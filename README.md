##  Objetivo

Desarrollar un sistema embebido capaz de monitorear en tiempo real la humedad del suelo y las condiciones ambientales para detectar estrés hídrico y prevenir el marchitamiento en cultivos.

---

##  Problema

La falta de monitoreo en tiempo real del nivel de humedad del suelo provoca pérdidas agrícolas, uso ineficiente del agua y disminución en la productividad de los cultivos.

---

## Componentes

- Microcontrolador (Arduino / ESP32)
- Sensor de humedad del suelo
- Sensor de temperatura y humedad (DHT11/DHT22)
- Pantalla LCD
- Buzzer
- LEDs indicadores
